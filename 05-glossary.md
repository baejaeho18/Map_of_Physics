## 고전물리학
| 용어 (한) | 용어 (영) | 한 줄 설명 |
|---|---|---|
| 운동 법칙 | Laws of Motion | 힘과 운동의 관계, $\vec F=m\vec a$ |
| 만유인력 | Universal Gravitation | 질량 있는 모든 물체는 서로 끌어당김 |
| 미적분 | Calculus | 변화를 다루는 수학 도구, 뉴턴이 함께 개발 |
| 해석역학 | Analytical Mechanics | 작용 최소화 관점의 역학(라그랑주·해밀턴) |
| 엔트로피 | Entropy | 무질서의 정도, 닫힌 계에서 증가 |
| 열역학 | Thermodynamics | 열·온도·에너지의 거시 법칙 |
| 통계역학 | Statistical Mechanics | 미시 입자 통계로 거시 열현상 설명 |
| 전자기학 | Electromagnetism | 전기·자기·빛을 하나로 묶음(맥스웰) |
| 혼돈 이론 | Chaos Theory | 초기 조건 민감성(나비효과) |

## 양자물리학
| 용어 (한) | 용어 (영) | 한 줄 설명 |
|---|---|---|
| 양자 | Quantum | 에너지를 주고받는 최소 덩어리 |
| 양자역학 | Quantum Mechanics | 확률·파동함수로 미시 세계 기술 |
| 파동함수 | Wave Function | 상태를 담은 함수, $\lvert\psi\rvert^2$=확률 |
| 중첩 | Superposition | 여러 상태에 동시에 걸쳐 있음 |
| 불확정성 원리 | Uncertainty Principle | 위치·운동량 동시 측정 한계 |
| 파동-입자 이중성 | Wave–Particle Duality | 빛·물질이 파동이자 입자 |
| 양자장론 | Quantum Field Theory (QFT) | 입자를 장의 들뜸으로 봄, 양자+특수상대성 |
| 표준모형 | Standard Model | 기본 입자와 세 힘의 체계 |
| 힉스 보손 | Higgs Boson | 질량을 부여하는 입자 |
| 반물질 | Antimatter | 전하가 반대인 입자 짝 |

## 상대성이론
| 용어 (한) | 용어 (영) | 한 줄 설명 |
|---|---|---|
| 특수 상대성 | Special Relativity | 빛 속도 불변 → 시간 지연·길이 수축 |
| 일반 상대성 | General Relativity | 중력 = 휘어진 시공간 |
| 시공간 | Spacetime | 시간과 공간을 하나로 본 4차원 무대 |
| 질량-에너지 등가 | Mass–Energy Equivalence | $E=mc^2$ |
| 블랙홀 | Black Hole | 빛도 못 빠져나오는 시공간 영역 |
| 사건의 지평선 | Event Horizon | 블랙홀의 되돌아올 수 없는 경계 |
| 우주론 | Cosmology | 우주 전체의 기원·구조·진화 |
| 빅뱅 | Big Bang | 우주가 한 점에서 시작해 팽창 |

## 무지의 심연
| 용어 (한) | 용어 (영) | 한 줄 설명 |
|---|---|---|
| 양자중력 | Quantum Gravity | 양자+중력 통합 이론(미완성) |
| 만물의 이론 | Theory of Everything | 모든 힘을 하나로 묶는 이론 |
| 끈 이론 | String Theory | 입자를 진동하는 끈으로 보는 가설 |
| 고리 양자중력 | Loop Quantum Gravity | 시공간이 알갱이로 되어 있다는 가설 |
| 암흑물질 | Dark Matter | 보이지 않는 정체불명의 질량 |
| 암흑에너지 | Dark Energy | 가속 팽창을 일으키는 정체불명 에너지 |
| 측정 문제 | Measurement Problem | 중첩이 왜 하나로 붕괴하는가 |
